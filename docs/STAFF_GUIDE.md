# LocalOps Planner Staff Guide

This guide is for staff users.

## First Login

Your manager will give you a username and temporary password.

Default temporary password:

```text
staff123
```

You must change your password before using the rota.

## Dashboard

Staff Dashboard shows:

- Your upcoming shifts
- Reminders
- Time-off status
- Tasks

Staff users only see their own shift information.

## My Shifts

Open My Shifts to see your upcoming shifts clearly.

For each shift you may see:

- Date
- Start time
- End time
- Notes
- Extra cover information
- Calendar buttons

## Time Off

Use Time Off to request holiday or unavailable days.

Workflow:

1. Choose start date.
2. Choose end date.
3. Add a reason.
4. Submit request.
5. Wait for admin approval or rejection.

Approved time off affects the rota totals.

## Tasks

Open Tasks to see shop jobs assigned to you or the team.

Move tasks as work progresses:

- Backlog
- Todo
- Process
- Done

Done tasks are hidden from the rota calendar.

## Reminders And Notifications

Open Reminders to see:

- Shift reminders
- Shift changes
- Notes added by admin
- Time-off approval/rejection messages

## Enable Phone Notifications

1. Log in on your phone.
2. If using iPhone, install the web app to the home screen first.
3. Open Account.
4. Press Enable Notifications.
5. Allow notifications.
6. Confirm the test notification appears.

Notes:

- Notifications are free.
- The app must be hosted on HTTPS.
- Browser and phone settings control notification sounds.

## Calendar Sync

You can add shifts to:

- Google Calendar
- Apple Calendar
- Outlook
- Phone calendar

Open My Shifts or Reminders and use the calendar buttons.

Account also includes a private calendar feed for upcoming shifts.

Keep this feed private.

## Change Password

Open Account to change your password.

Use a password that is not shared with other systems.
